# Thin Section Environment

The Thin Section environment offers various analyses and image processing of rock thin sections. With this environment, you can perform image loading and registration up to different types of segmentation and results analysis.

## Sections

The GeoSlicer Thin Section environment is organized into several modules, each dedicated to a specific set of tasks.

*   **[Thin Section Loader](/ThinSection/Loader/ThinSectionLoader.md):** Loads thin section images.
*   **[QEMSCAN Loader](/ThinSection/Loader/QemscanLoader.md):** Loads QEMSCAN images.
*   **[Crop](/ThinSection/Crop/Crop.md):** Allows cropping of thin section images.
*   **[Filters](/ThinSection/Filters/ImageTools.md):** Provides filters to enhance the image.
*   **[Register](/ThinSection/Register/Register.md):** Aligns and registers images in a common coordinate system.
*   **[Segmentation](/ThinSection/Segmentation/Segmentation.md):** Perform image segmentation using [automatic](/ThinSection/Segmentation/Segmentation.md#automatic-thin-section-segmentation) or [manual](/ThinSection/Segmentation/Segmentation.md#manual-segmentation) methods to separate different phases and features in your rock samples.
*   **[Flows](/ThinSection/Flows/Flows.md):** Contains predefined workflows for different types of analysis.
*   **[Multiple Image Analysis](/ThinSection/MultipleImageAnalysis/MultipleImageAnalysis.md):** Allows the analysis of multiple images simultaneously.
*   **[Export](/ThinSection/Export/Export.md):** Offers options to export the results.
*   **[More tools](/ThinSection/MoreTools/MoreTools.md):** Explore additional tools for analysis and visualization of results.
*   **[Import](/ThinSection/Import/Import.md):** Imports data from standardized sources.

## What can you do?

With the GeoSlicer Thin Section environment, you can:

*   **Visualize and analyze thin section images.**
*   **Apply a wide range of image processing filters.**
*   **Segment images to identify different phases and features.**
*   **Perform multiple image analyses.**
*   **Export results for reports and other applications.**
