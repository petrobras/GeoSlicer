## Segment Editor

The *Segment editor* module to segment an image, as described in the steps below:

{{ video("thin_section_manual_segmentation.webm", caption="Video: Manual Segmentation") }}

1. Enter the manual segmentation section of the environment.
2. Select *create new segmentation* in *Output segmentation*.
3. Select a reference image in *Input image*.
4. Click *Add* to add the number of segments.
5. The visibility (eye icon), color, and name of the segments can be changed in the segments table.
6. Select the desired effect from the left-side menu next to the segments table; in the video, *color threshold* and *paint* are used.
7. Parametrize the desired effect and choose the region to be segmented.
8. Click *Apply* and wait for completion. A segmentation node will appear, and its visualization can be changed in the Explorer.
