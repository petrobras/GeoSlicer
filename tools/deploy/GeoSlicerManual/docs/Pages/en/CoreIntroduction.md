# Core Environment

The Core environment is dedicated to the analysis of drilling core samples, allowing for the visualization and processing of core images. With this environment, you can perform a wide range of tasks, from loading photographic images of core samples to segmentation and export of results.

## Sections

The GeoSlicer Core environment is organized into several modules, each dedicated to a specific set of tasks. Click on a module to learn more about its functionalities:

*   **[Import](/Core/Multicore.md):** Designed to import and process core images.
*   **[Core Photograph Loader](/Core/CorePhotographLoader.md):** Load photographic images of core samples for analysis.
*   **[Crop](/Core/Crop.md):** Tools for cropping and adjusting core images.
*   **[Transforms](/Core/MulticoreTransforms.md):** Apply geometric transformations.
*   **[Segmentation](/Core/Segmentation/Segmentation.md):** Perform image segmentation for feature identification.
*   **[Core Inpaint](/Core/CoreInpaint.md):** Fill in missing or damaged regions in images.
*   **[Export](/Core/MulticoreExport.md):** Export processed data and analysis results.

## What can you do?

With the GeoSlicer Core environment, you can:

*   **Visualize and analyze drilling core images.**
*   **Apply a wide range of image processing filters and transformations.**
*   **Segment images to identify geological features.**
*   **Inspect and refine segmentation results.**
*   **Export results for use in other software or for reports.**
